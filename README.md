## 🚀 NexGen - Cutting-Edge Digital Solutions Agency

NexGen is a modern, high-performance landing page for a Digital Solutions Agency. Built with the latest web technologies, including React 19 and Tailwind CSS v4, it features a sleek dark-themed UI, smooth animations, and a fully responsive layout designed to drive
success.

### ✨ Key Features

- Modern UI/UX: A professional dark-mode aesthetic with high-quality components.
- Highly Responsive: Optimized for all devices, from mobile phones to large desktops.
- Dynamic Sections: Includes Hero, Services, Portfolio, Testimonials, and FAQ.
- Form Validation: A robust contact form powered by Formik and Yup.
- Smooth Motion: Interactive elements and marquees using React Fast Marquee.
- Clean Architecture: Modular component structure for easy scalability.

### 🛠️ Tech Stack

- Core: React 19 & Vite
- Styling: Tailwind CSS v4
- Routing: React Router 7
- Forms: Formik & Yup
- Icons: FontAwesome (Brands, Regular, Solid)
- Linting: ESLint

### 📂 Project Structure

```bash
src/
├── assets/         # Images, icons, and static assets
├── components/
│   ├── layout/     # Global components like Navbar and Footer
│   ├── sections/   # Individual sections of the page (Hero, Services, etc.)
│   └── UI/         # Reusable UI elements (Buttons, Inputs, Cards)
├── data/           # Static data files for easy content management
└── pages/          # Main application pages and route views
```

### 🚀 Getting Started

Follow these steps to run the project locally:

1. Clone the repository:

   ```bash
   git clone https://github.com
   ```

2. Navigate to the project directory:

   ```bash
   cd nexgen-project
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

### 📜 Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Compiles the project for production.
- `npm run lint`: Checks for code quality issues using ESLint.
- `npm run preview`: Previews the production build locally.
